Deterermine the result of the following expressions.  
```cpp
cout << -30 * 3 + 21 / 5 << endl;   // (a)
cout << -30 + 3 * 21 / 5 << endl;   // (b)
cout <<  30 / 3 * 21 % 5 << endl;   // (c)
cout << -30 / 3 * 21 % 4 << endl;   // (d)
```
These expressions will evaluate to the following numbers.  
```cpp
-86
-18
0
-2
```
